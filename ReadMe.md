### Projects

**[Emoji Explorer](https://github.com/realastronics/emoji-explorer)**
&nbsp;`Kotlin` &nbsp;`ARCore` &nbsp;`Firebase` &nbsp;`GPS`

Pokémon Go–style AR game anchored to the BMU campus. Players explore physical space to find and capture emoji objects through their camera. Built for ACM BMU's GLITCH 2.0 techfest — played by real people at a real event. Team scoring, power-ups, sabotage, live leaderboard.

<br/>

**[Hiring Pipeline Automation](https://github.com/realastronics/Hiring-Pipeline-Automation)**
&nbsp;`JavaScript` &nbsp;`Gmail API` &nbsp;`Google Calendar` &nbsp;`OAuth`

AI-powered hiring dashboard for lean HR teams. Screens resumes in seconds, sends personalised outreach from your own Gmail, books interviews directly on Google Calendar. Currently in private beta. → [hiring-pipeline-automation.vercel.app](https://hiring-pipeline-automation.vercel.app)

<br/>

**[Utsav](https://github.com/realastronics/Application-Utsav)**
&nbsp;`Java` &nbsp;`Android` &nbsp;`Firebase`

Android marketplace connecting event hosts with professional managers. Hosts filter by location and budget; managers build portfolios and handle requests from a dedicated dashboard. In-app chat throughout.

<br/>

**[Flappy Bird — Hand Gesture Edition](https://github.com/realastronics/FlappyBird-Using-Visual-Input)**
&nbsp;`Python` &nbsp;`MediaPipe` &nbsp;`Pygame`

Fist to jump. Pinch to flap. Built for ACM BMU's club fair as a tactile intro to computer vision — freshers could immediately tweak `config.py` and see the physics change.

<br/>

**[GameLinked](https://github.com/realastronics/GameLinked)**
&nbsp;`Node.js` &nbsp;`Express` &nbsp;`PostgreSQL` &nbsp;`Prisma`

LinkedIn for gamers, built backend-first. REST API, Prisma ORM, clean service/controller separation. Started because social networks are interesting as systems problems.
